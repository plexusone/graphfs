## [v0.1.0] - 2026-04-09

### Highlights

- Git-friendly graph database with one-file-per-entity storage for minimal diffs
- Edge confidence system supporting EXTRACTED, INFERRED, and AMBIGUOUS relationships

### Added

- `graph.Node` type with ID, Type, Label, and extensible Attrs map
- `graph.Edge` type with From, To, Type, Confidence, ConfidenceScore, and Attrs
- `graph.Graph` type with in-memory node/edge collections and CRUD operations
- `graph.Confidence` enum: EXTRACTED (AST-derived), INFERRED (LLM), AMBIGUOUS (uncertain)
- Edge type constants: calls, imports, implements, extends, uses, contains, depends_on, references
- `store.FSStore` for filesystem-backed persistence with `nodes/` and `edges/` directories
- `store.Store` interface for pluggable storage backends
- One-file-per-entity storage: `nodes/{id}.json` and `edges/{from}--{type}-->{to}.json`
- `format.MarshalCanonical` and `UnmarshalCanonical` for deterministic JSON output
- Graph operations: `SaveGraph`, `ListNodes`, `ListEdges`, `GetNode`, `GetEdge`
- `schema.Validate` for validating graph structure
